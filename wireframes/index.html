Estructura y layout de componentes principales del sistema

📋 Acerca de estos wireframes:
Estos wireframes muestran la estructura visual real basada en los componentes React del proyecto. Utilizan únicamente colores en escala de grises (#ffffff, #f0f0f0, #cccccc, #666666, #333333) sin elementos de branding, logos a color, o imágenes reales. Cada wireframe indica claramente dónde irían los elementos principales.
